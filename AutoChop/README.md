# [JR] Auto Chop

Automation plugins for runelite

# Discord

https://discord.gg/PwRB5yt8KH

**How to setup**:

    - Axe Equipped or in inventory
    - Be at tree location:
        - Tree: GE
        - Willow: Draynor
        - Teak: Soul Wars
        - Maple: Seers
        - Yew: Seers
        - Magic: Hosidius

**Supported Forestry Events**:

    - Rising Roots
    - Woodcutting Leprechaun
    - Beehive
    - Friendly Ent
    - Poachers
    - Enchantment Ritual
    - Pheasant Control

**Things To Know**:
- PLUGIN PathingTester MUST BE TURNED ON
